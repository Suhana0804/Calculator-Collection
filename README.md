
# Loan Calculator

This is a simple Loan Calculator made using Python and Tkinter.

The program calculates:
- Total loan amount after interest
- Monthly payment amount

The user enters:
- Annual interest rate
- Number of years
- Loan amount

After clicking the button, the program displays the total amount and monthly payment.

## Features

- GUI made using Tkinter
- Loan payment calculation
- Monthly payment calculation
- Simple user interface

## Technologies Used

- Python
- Tkinter

## How to Run

Run the Python file:


python loan_calculator.py
