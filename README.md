# Calculator Collection

This repository contains simple calculator projects made using Python and Tkinter.
# Loan Calculator

This is a simple Loan Calculator made using Python and Tkinter.

The program calculates:
- Total loan amount after interest
- Monthly payment amount

The user enters:
- Annual interest rate
- Number of years
- Loan amount

After clicking the button, the program displays the total amount and monthly payment.

## Features

- GUI made using Tkinter
- Loan payment calculation
- Monthly payment calculation
- Simple user interface

## Technologies Used

- Python
- Tkinter

## How to Run

Run the Python file:


python loan_calculator.py


## What I Learned

- Tkinter GUI basics
- Labels, buttons, and entry boxes
- Grid layout system
- Functions in Python
- Taking user input
- Basic financial calculations
